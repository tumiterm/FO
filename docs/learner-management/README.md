# Facilitator Learner Management — Visual Journey Map

This folder contains a non-technical, end-to-end illustration of the Forek Online facilitator learner-management workflow.

## Open or export the visual

1. Open [`index.html`](index.html) in any modern browser.
2. Select **Download editable SVG** to use the vector artwork in presentations, design tools, documents, or websites.
3. Select **Print / Save as PDF** to create a PDF suitable for email, workshops, or approval packs.

The SVG can be scaled to any size without becoming blurry and can be opened by common tools such as a web browser, PowerPoint, Figma, Inkscape, Illustrator, or diagrams.net.

## What the illustration covers

- The facilitator, learner, and administrator roles.
- Selecting learners and the non-blocking warning when another facilitator is already linked.
- The facilitator's **My Students** operational list and retained removal history.
- Coloured learning groups, optional notes, multiple group membership, and ungrouped learner alerts.
- In-app notifications and the WhatsApp queue/log skeleton.
- Group-based attendance, Draft and Submitted states, attendance statuses, comments, and correction history.
- Learner absence alerts.
- Administrator coverage, allocation-gap, group, attendance, communication, and audit views.
- Safeguards and planned extension points.
